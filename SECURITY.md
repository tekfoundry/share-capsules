# Security Policy

## Experimental status

Share Capsules, Capsule, and CTX are under active development and have not completed the security review required for a production release. Do not rely on the current implementation to protect highly sensitive content.

## Reporting a vulnerability

Please do not open a public issue for a suspected vulnerability or exposed secret.

Use GitHub private vulnerability reporting when it is enabled for the repository. Otherwise, email [info@tekfoundry.com](mailto:info@tekfoundry.com) with the subject `Share Capsules security report`.

Include only the information needed to reproduce and assess the issue:

- Affected component and version or commit
- Reproduction steps or proof of concept
- Expected and observed behavior
- Potential security or privacy impact
- Any suggested mitigation

Do not include real creator content, credentials, private keys, recovery material, access tokens, identity documents, or unrelated personal data. Use synthetic test material whenever possible.

TekFoundry will acknowledge a report, investigate it, coordinate remediation and disclosure when appropriate, and credit reporters who request attribution and act in good faith.

## Supported versions

There is no supported production release yet. Security fixes currently target the active development branch.
