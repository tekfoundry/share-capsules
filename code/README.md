# Share Capsules application

This directory contains the Laravel control plane and TypeScript workspaces for the Share Capsules reference implementation of Capsule and CTX.

Run development and quality commands from the repository root through `./_infra/kit`; dependencies and tooling execute inside the project containers. See the [root README](../README.md) for setup, project status, security guidance, sponsorship, and licensing.

The application is experimental. It must not yet be relied upon to protect sensitive or irreplaceable content.
